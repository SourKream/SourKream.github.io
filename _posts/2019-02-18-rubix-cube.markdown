You can use these keys to interact with the game,

- `E` : Enter edit mode.
- `A` `S` `D` : Rotate planes while in edit mode.
- `UP_ARROW` `DOWN_ARROW` : Shift selected edit planes.

You can also use the mouse to rotate the cube around. Scroll to zoom.
